## sh
```
git clone https://github.com/chubbyhippo/init.lua.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```
```
curl https://raw.githubusercontent.com/chubbyhippo/init.lua/main/init.lua -o "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim/init.lua
```
